How to Play

🎮 Controls:

  • WASD or Arrow Keys - Move
  • Mouse - Aim
  • Space - Start Next Wave
  • R - Manual Reload
  • Ctrl+S - Save Game
  • Ctrl+L - Load Game

⚔️ Combat Tips:

  • Weapons auto-attack the closest enemy
  • Sniper targets highest HP enemies
  • Shotgun is great for groups
  • Crossbow pierces through enemies

🏥 Survival Tips:

  • Place Healing Towers to regenerate HP
  • Use Bombs for emergency crowd control
  • Merge identical weapons to tier up
  • Choose stat buffs wisely after each wave
